# Unsplash Gallery

Uma galeria de imagens inspirada no Pinterest, utilizando a API do Unsplash.

## Tecnologias

- Next.js 14
- TypeScript
- Tailwind CSS
- shadcn/ui
- Framer Motion
- React Masonry CSS
- React Infinite Scroll

## Funcionalidades

- Busca de imagens
- Layout em masonry
- Scroll infinito
- Modal de detalhes
- Sistema de likes
- Responsivo
- Efeitos visuais
